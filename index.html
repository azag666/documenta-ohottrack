Crie seu Checkout com Inteligência Artificial

Não sabe programar? Sem problemas. Use este guia de API junto com o seu modelo de IA favorito (ChatGPT, Claude, Cursor) para gerar um checkout completo e integrado à HotTrack.

💡 Como funciona?

Abaixo você encontrará a especificação exata das rotas de pagamento da HotTrack. No final da página, preparamos um Prompt Mestre. Basta copiar o prompt, colar na sua IA de preferência e assistir seu checkout PIX de alta conversão ser gerado automaticamente.

Referência de API para fornecer à IA

Passo 1

Capturar Click ID (Resolve Click)

Esta rota é usada para ler os parâmetros de inicialização do Telegram Mini App ou do checkout e gerar o identificador único de rastreamento (click_id).

POST https://hot-track.com/api/miniapp/resolve-click
JSON Body Obligatório
{
    "initData": "string (Dados de inicialização vindos do Telegram)",
    "checkoutId": "string (O ID do seu checkout na HotTrack)"
}
Resposta Retornada (Response)
{
    "click_id": "clk_123456789abc"
}
Passo 2

Gerar Cobrança PIX & Ativar Tracking

Cria uma nova ordem de pagamento em PIX utilizando o click_id obtido no passo anterior. O valor deve sempre ser enviado em centavos.

POST https://hot-track.com/api/pix/generate
  • Header Requerido: x-api-key: Sua chave secreta de API HotTrack.
  • Header Requerido: Content-Type: application/json
JSON Body Obligatório
{
    "click_id": "clk_123456789abc",
    "value_cents": 1999 // Exemplo para R$ 19,99 (Sempre em centavos)
}
Resposta Retornada (Response)
{
    "transaction_id": "tx_987654321xyz",
    "qr_code_text": "00020101021226870014br.gov.bcb.pix..." // Código Copia e Cola
}
Passo 3

Verificar Status do Pagamento

Consulta em tempo real se a cobrança PIX já foi liquidada pelo cliente. Ideal para criar loops de checagem automática.

GET https://hot-track.com/api/pix/status/{transaction_id}
  • Header Requerido: x-api-key: Sua chave secreta de API HotTrack.
Resposta Retornada (Response)
{
    "status": "paid" // Pode retornar "pending" se ainda não foi pago
}

O Prompt Mestre da IA

Copie o prompt abaixo integralmente e cole no seu chat com a IA (Claude ou ChatGPT). Lembre-se de substituir os placeholders marcados com [] com os dados do seu produto.

🚀 Prompt para Copiar e Colar

Atue como um desenvolvedor especialista em UI/UX FinTech sênior. Preciso que você crie um código de checkout em arquivo único (HTML, CSS embutido e JavaScript puro) para vender o meu produto digital: [NOME DO SEU PRODUTO]. O design deve seguir o padrão estético de plataformas de alta tecnologia como Stripe e Linear: tema escuro super limpo, fontes elegantes (Inter), espaçamentos bem definidos, bordas discretas e sem elementos visuais poluídos ou cores neon exageradas. O checkout deve realizar a integração completa com a API da HotTrack seguindo rigorosamente este fluxo técnico: 1. Ao carregar a página, capture o 'initData' (do ambiente do Telegram) e envie um POST para `https://hot-track.com/api/miniapp/resolve-click` contendo o JSON com o 'initData' e o meu 'checkoutId' fixo que é: [INSIRA_SEU_CHECKOUT_ID_AQUI]. Guarde o 'click_id' retornado. 2. Em seguida, acione automaticamente (ou via clique de botão de compra) a rota de geração do PIX enviando um POST para `https://hot-track.com/api/pix/generate`. Passe o Header 'x-api-key' com o valor [SUA_API_KEY_AQUI] e no body envie o 'click_id' capturado e o valor 'value_cents' fixado em [VALOR_EM_CENTAVOS_AQUI, ex: 1999 para R$19,99]. 3. Quando receber o 'qr_code_text' e o 'transaction_id', exiba uma interface limpa contendo: - Um campo de texto com o código PIX Copia e Cola. - Um botão proeminente de "Copiar Código PIX" que jogue o texto para a área de transferência e dê um feedback visual de "Copiado!". - Um aviso discreto informando que o código expira em breve. 4. Inicie imediatamente após a geração um sistema de polling (verificação cíclica) a cada 4 segundos fazendo uma requisição GET para `https://hot-track.com/api/pix/status/{transaction_id}` passando o Header 'x-api-key'. Se o retorno for `"status": "paid"`, pare a verificação e altere a tela de forma fluida para uma linda mensagem de sucesso de "Pagamento Confirmado!". Gere o arquivo completo estruturado pronto para uso.
